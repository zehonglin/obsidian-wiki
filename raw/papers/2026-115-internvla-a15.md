# InternVLA-A1.5: 统一理解、潜在前瞻与动作

- **来源**: https://arxiv.org/html/2607.04988v1
- **日期**: 2026-07-10
- **方向**: 具身智能VLA
- **Topic匹配**: T008(3D/空间感知), T010(组合泛化)
- **相关度**: 0.88

## 核心内容

基于原生VLM骨干网络(Qwen3.5)构建策略，持续训练VQA和子任务预测，附加轻量级统一专家进行连续动作生成。

## 主要创新

1. **潜在前瞻(Latent Foresight)**: 将未来预测重新构造为潜在查询问题，使用少量可学习的前瞻token在冻结的视频生成模型监督下，将任务相关的未来压缩为紧凑的潜在码
2. 策略继承世界模型的动力学先验，无需学习像素级生成
3. **训练规模**: 120万机器人episode + 300万多模态样本
4. **性能**: 在6个仿真基准上取得最佳整体结果

## 关键架构

- 基于Qwen3.5原生VLM骨干
- 保持VQA + 子任务预测 + 离散动作token目标
- 轻量级统一专家用于连续动作生成

## 与我们研究的关联

- **T008**: 潜在前瞻token提供空间-时序感知
- **T010**: 组合泛化能力，大规模预训练策略
- 与STARRY(2026-018)、MemoryWAM(2026-060)等世界模型方向论文相关
