# Pre-VLA: Preemptive Runtime Verification for Reliable VLA and World-Model Rollouts

- **arXiv**: 2605.22446
- **Authors**: Zhen Sun, Yongjian Guo, Haoran Sun, Luqiao Wang, Wei Lu, Jiachi Ji, Shengzhe Ji, Junwu Xiong, Zhijun Meng
- **Date**: 2025-05
- **Direction**: VLA

## Summary
Pre-VLA提出了一种统一的运行时验证架构，在VLA候选动作执行物理操作或进入世界模型想象之前，进行先发式动作有效性评估。核心设计包括：
1. 基于WorldVLA的高效多模态骨干网络，通过模态感知池化和轻量级双分支头来预测安全置信度和critic优势分数
2. 多任务训练目标：Focal分类 + 优势回归 + 软阈值校准，解决类别不平衡和边界决策不稳定问题
3. 双模式先发式重采样调度器，在有限计算预算下过滤低质量动作并触发自适应重采样

## Key Results
- LIBERO基准：平均成功率从30.79%提升至37.62%（+6.83pp，基于RynnVLA-002）
- 动作验证F1=0.8303，准确率=0.9542，无效动作误放率仅0.0200
- 单次前向验证时间约183.9ms，计算开销可控
- 有效缓解世界模型想象中的误差累积
